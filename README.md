# 3分钟零门槛注册ChatGPT流程
前两年的 ChatGPT 注册需要海外手机号验证，劝退了不少人，但现在情况已经大不相同。
如今，只要搞定适配的网络和“邮箱这两件事，你就已经成功了 90%。

第一步：首先准备工作
在开始之前，要准备好这两样东西：
1.畅通的网络环境
1.需要能够稳定访问 chatgpt.com 的网络节点（建议选择美国、新加坡或日本节点）。节点推荐：https://pikayun.com/
2.一个合适的邮箱
Gmail（谷歌）或 Outlook（微软）是目前最合适的，国内的很多邮箱：QQ、163、126等被拒之门外，无法注册

第二步：搞定邮箱（首选 Outlook）
如果你已经有 Gmail 或 Outlook，请直接跳至第三步。 如果你是纯新手，Outlook 是最佳选择，因为它不需要手机号验证，注册过程非常方便
注册 Outlook 指南：
访问官网： 点击 Microsoft Outlook 注册页面，选择 “创建免费账户”。
设置账号： 输入你喜欢的邮箱前缀，点击“下一步”。
设置密码： 创建一个安全的密码。
填写信息：
国家/地区： 建议选 美国 (United States)（有助于后续服务兼容）。
生日： 建议填写成年人的出生日期。
人机验证： 完成简单的图形验证。

第三步：注册 ChatGPT
万事俱备，现在的注册流程仅需 3 分钟。
1.进入官网 打开浏览器（推荐 Chrome 无痕模式），访问 chatgpt.com。
2.开始注册 点击页面右上角的 "Sign up"（注册）按钮。
填写信息
1. 输入你刚才准备好的 Outlook 邮箱地址，点击 "Continue"。
2.设置密码 输入密码，要求至少 12 位字符，点击 "Continue"。
3.邮箱验证 OpenAI 会向你的邮箱发送一封验证邮件。
登录邮箱，找到邮件，将里面的 6 位数验证码 填回 ChatGPT 的注册页面。
完善资料 填写你的姓名（拼音即可）和生日。
注册成功后，你获得了免费的chatgpt账号，当然也可以升级会员，现在的免费账号依然强大，你通常可以体验到：
基础模型对话：流畅的问答与写作。
历史记录与搜索：随时回顾过往的灵感。
语音对话：像打电话一样和 AI 聊天。
更多高级功能：根据 OpenAI 的更新，你甚至有机会体验到 GPTs 商店、Sora（视频生成）等前沿功能。
你可以先体验免费版的，如果免费的功能不足以满足你的需求量，你可以开通会员。
